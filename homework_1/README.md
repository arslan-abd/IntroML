# Отчет по homework_1 по анализу данных с использованием Pandas

## Описание проекта
Анализ датасета "adult_data.csv", содержащего демографические данные и информацию о доходах населения. Проект направлен на освоение практических навыков работы с библиотекой Pandas для решения реальных аналитических задач.

## Цели проекта
1. Освоение базовых операций с данными в Pandas
2. Разработка навыков фильтрации, агрегации и статистического анализа
3. Освоение визуализации данных с помощью Matplotlib и Seaborn
4. Получение практического опыта работы с реальными данными
